üí¨ Section 2: Prompt Engineering

üí° The Perfect Prompt ‚ú®

The quality of your output depends heavily on the quality of your input. Here are some keys to a great prompt:

  • ‚û°Ô∏è Be Specific: Provide details and context.
  • ‚û°Ô∏è Define the Persona: "Act as a senior software engineer..."
  • ‚û°Ô∏è Provide Examples: Show the format you want.
  • ‚û°Ô∏è Iterate: Refine your prompt based on the output.

üìö Prompt Example: Explaining a Complex Topic

Using a persona and clear instructions helps the AI tailor its explanation.


Act as a computer science professor teaching an introductory course.

Explain how a CPU works to a first-year university student.

Use an analogy to make it easier to understand.
Keep the explanation concise, no more than 3 paragraphs.

üí° Prompt Example: Brainstorming Creative Ideas

AI can be a powerful tool for creative brainstorming, providing diverse perspectives.

The slogans should be:
- Short and memorable.
- Highlight sustainability.
- Appeal to young adults.

üíª Prompt Example: Generating Code

AI can assist developers by generating code snippets or even entire scripts.


Act as a Python game developer.

Write the code for a simple text-based adventure game.

The game should have:
1. A player who can move between rooms.
2. At least three rooms: "Cave Entrance", "Dark Tunnel", "Treasure Room".
3. The player should be able to type commands like "go north", "go south".
4. The goal is to find the treasure in the "Treasure Room".
5. The code should be a single Python script.

üéÆ Prompt Example: Creating a Simple Pygame Game

Let's create a basic game inspired by Geometry Dash using Pygame.


Act as an experienced Pygame developer.

Create a simple 2D endless runner game in Pygame.

The game should feature:
1.  A white square player that can jump.
2.  Green square obstacles that move from right to left.
3.  Collision detection between the player and obstacles, ending the game.
4.  A score counter that increases over time.
5.  Basic game loop, event handling, and rendering.
6.  The code should be a single Python file.

üòÇ Prompt Example: Creating a High School Joke Website

Let's create a fun and engaging website tailored for a high school audience!


Act as a creative web developer specializing in content for high schoolers.

Create a single HTML file for a simple website called "High School Humor".

The website should have:
1.  A main heading "Welcome to High School Humor!".
2.  At least 5 short, witty, and appropriate jokes for high schoolers.
3.  Each joke should have a question and an answer, clearly separated.
4.  Use basic HTML and CSS for a clean and readable design.
5.  All CSS should be included in a `<style>` tag within the head of the HTML document.

‚ú® Perfecting the Image Prompt ‚úçÔ∏è

For image generation, detail is key. The model can't see what's in your head.

  • ‚û°Ô∏è Subject: What is the main focus? Be very descriptive.
  • ‚û°Ô∏è Style: Photorealistic, cartoon, oil painting, 3D render?
  • ‚û°Ô∏è Composition: Close-up, wide shot, portrait?
  • ‚û°Ô∏è Lighting: Soft morning light, dramatic cinematic lighting, neon glow?
  • ‚û°Ô∏è Keywords: Use terms like "masterpiece", "highly detailed", "8K".

üñºÔ∏è Image Prompt Examples

Simple Prompt üëé

"An astronaut"

Result: A generic, likely simple drawing of an astronaut.

Detailed Prompt üëç

"Photorealistic shot of an astronaut on a vibrant, alien planet with two suns, dramatic lighting, 1970s sci-fi film style, highly detailed."

Result: A rich, cinematic, and specific image.

üé® Prompt Example: Creating a Fun Art Generator Website

AI agents can help build complex, interactive applications like a website that generates unique art based on user input!


Act as a full-stack web developer specializing in creative AI applications.

Your goal is to create a web application that generates unique abstract art based on user-selected parameters.

The application should have:
1.  An HTML structure with a canvas element for rendering art and control elements (buttons, sliders).
2.  CSS for a modern, minimalist design.
3.  JavaScript to handle user input and draw abstract art on the canvas.
4.  Parameters for users to adjust: color palette, shape complexity, and animation speed.
5.  A "Generate Art" button that creates new art based on current parameters.
6.  The art generation logic should be simple, using basic geometric shapes and random colors.
